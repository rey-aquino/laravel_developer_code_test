## Installation Guide

- Clone repo
- Composer install
- `composer install`
- NPM install
- `composer install`
- Copy .env.example file
- `cp .env.example .env`
- Create database and update .env file
  `DB_DATABASE=database_name`
- Generate key
- `php artisan key:generate`
- Generate key
- `php artisan key:generate`
- Migrate database
- `php artisan migrate`
